# Step 4: Accounts & Properties Flow - Complete In-Depth Analysis

## Overview
Step 4 involves adding and managing all assets and liabilities owned by the user. This includes real estate, vehicles, gadgets, jewellery, investments, liabilities, personal business, and household items. Each asset type has its own flow for adding details, assigning value, and determining inheritance distribution.

## Main Properties & Accounts Screen

### Screen: Properties & Accounts (Main)
**Purpose:** Entry point showing all asset categories

**Elements:**
- Header: Back arrow, "Properties & accounts"
- Subtitle: "Add stuff that you own. Why it's important" (with expandable link)
- **Informational Cards (3 cards explaining benefits):**
  1. "Transfer ownership hassle free" - "No court visit and challenge from others"
  2. "Customise the share among heirs" - "Customise among heirs or add a new one"
  3. "Track your total wealth" - "Be aware and plan future to meet needs"
- **Asset Categories List:**
  - Each category has:
    - Grey placeholder icon (left)
    - Category name (bold)

    - "Add" or "View" button (green, right)

**Categories:**
1. **Real-estate** - "Purchased flat, property"

## Common Patterns Across All Categories

### 1. Inheritance Distribution Modal
**Appears when:** User taps "Transfer to" field

**Elements:**
- Modal title: "Who gets to own [asset] once you are no more"
- Description explaining default legal distribution
- Profile pictures of eligible heirs
- Green checkmarks on selected heirs
- "Got It" or "Finish" button

**Logic:**
- Default: All heirs & mother (if applicable)
- Can customize to specific heirs
- Can add new person if needed

### 2. Value Selection
**Pattern:** Dropdown with value ranges
- Real estate: "₹50 lakhs - ₹1 Crore", etc.
- Vehicles: "₹4 lakhs - ₹5 lakhs", etc.
- Other assets: Similar range-based selection

### 3. Loan/Installment Toggle
**Pattern:** Toggle switch for assets on loan/EMI
- When ON: Shows loan details fields
- When OFF: Hides loan fields

### 4. Photo Capture
**Pattern:** Camera integration for physical assets
- Real estate: Property photos
- Vehicles: Vehicle photos
- Gadgets: Front and back photos
- Jewellery: Individual item photos

### 5. List View Screens
**Pattern:** "My [Category]" screens
- Show all added items
- Each item has "Edit" button
- "Add another [item]" button
- "Done for now" link

### 6. Auto-track vs Manual Entry
**Pattern:** Two entry methods
- Auto-track: Connect to external services (credit reports, investment platforms)
- Manual entry: User enters details manually

## Step Completion Criteria

Step 4 is considered complete when:
- User has added at least one asset OR
- User has explicitly marked step as complete
- All added assets have:
  - Complete details
  - Estimated value

- `POST /wills/:willId/assets/vehicles` - Add vehicle

- `POST /wills/:willId/assets/jewellery` - Add jewellery

- `PATCH /wills/:willId/assets/:assetId` - Update asset
