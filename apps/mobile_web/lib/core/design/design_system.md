# Mywasiyat Design System

## Design Principles

1. **Calm**: Soothing colors, ample whitespace, gentle animations
2. **Premium**: High-quality typography, refined components, attention to detail
3. **Minimal**: Clean layouts, essential elements only, no clutter
4. **Milestone-driven**: Clear progress indicators, step-by-step guidance

## Color Palette

### Primary Colors (Calm & Professional)
- **Primary**: `#2563EB` (Trust Blue) - Main actions, links
- **Primary Dark**: `#1E40AF` - Hover states
- **Primary Light**: `#DBEAFE` - Backgrounds, highlights

### Secondary Colors
- **Secondary**: `#10B981` (Success Green) - Completed steps, success states
- **Accent**: `#F59E0B` (Warm Amber) - Warnings, attention needed
- **Error**: `#EF4444` (Soft Red) - Errors, critical actions

### Neutral Colors
- **Background**: `#FAFBFC` (Off-white) - Main background
- **Surface**: `#FFFFFF` - Cards, inputs
- **Border**: `#E5E7EB` - Dividers, borders
- **Text Primary**: `#111827` - Main text
- **Text Secondary**: `#6B7280` - Secondary text
- **Text Tertiary**: `#9CA3AF` - Placeholders, hints

## Typography

### Font Family
- **Primary**: Inter (or SF Pro on iOS, Roboto on Android)
- **Display**: Playfair Display (for headings, optional)

### Scale
- **Display Large**: 32px, Bold - Hero titles
- **Display Medium**: 28px, Bold - Page titles
- **Headline**: 24px, SemiBold - Section headers
- **Title**: 20px, SemiBold - Card titles
- **Body Large**: 16px, Regular - Body text
- **Body Medium**: 14px, Regular - Secondary text
- **Body Small**: 12px, Regular - Captions, hints

## Spacing System

- **XS**: 4px
- **S**: 8px
- **M**: 16px
- **L**: 24px
- **XL**: 32px
- **XXL**: 48px
- **XXXL**: 64px

## Border Radius

- **S**: 4px - Small elements
- **M**: 8px - Buttons, inputs
- **L**: 12px - Cards
- **XL**: 16px - Large cards
- **Full**: 999px - Pills, badges

## Shadows

- **Small**: `0 1px 2px rgba(0,0,0,0.05)` - Subtle elevation
- **Medium**: `0 4px 6px rgba(0,0,0,0.07)` - Cards
- **Large**: `0 10px 15px rgba(0,0,0,0.1)` - Modals

## Components

### Buttons
- **Primary**: Blue background, white text, 8px radius
- **Secondary**: White background, blue text, blue border
- **Text**: Transparent, blue text
- **Disabled**: Gray background, 50% opacity

### Cards
- White background, 12px radius, medium shadow
- Padding: 16px-24px
- Hover: Slight elevation increase

### Inputs
- White background, 8px radius
- Border: 1px solid #E5E7EB
- Focus: Blue border, 2px
- Error: Red border

### Progress Stepper
- Circular indicators with numbers/checkmarks
- Connecting lines between steps
- Color coding: Gray (not started), Blue (active), Green (completed), Amber (needs attention)

## Animations

- **Duration**: 200-300ms for most interactions
- **Easing**: Ease-in-out
- **Micro-interactions**: Subtle hover effects, button presses
- **Page transitions**: Slide or fade (200ms)

## Responsive Breakpoints

- **Mobile**: < 600px
- **Tablet**: 600px - 1024px
- **Desktop**: > 1024px
