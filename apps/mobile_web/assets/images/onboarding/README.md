# Onboarding Images

This directory contains the hero images for the onboarding flow screens.

## Required Images

Please add the following image files to this directory:

1. **calendar.jpg** - Calendar image for "When were you born?" screen
   - Description: April 2021 desk calendar on a wooden windowsill
   - Recommended size: 720x200px or similar aspect ratio

2. **couple.jpg** - Couple image for "Do you have a spouse?" screen
   - Description: Young couple on a beach, man giving woman a piggyback ride
   - Recommended size: 720x200px or similar aspect ratio

3. **family.jpg** - Family image for "Do you have any children?" screen
   - Description: Asian family of three (father, mother, daughter) at a dining table
   - Recommended size: 720x200px or similar aspect ratio

4. **child.jpg** - Child image for "Are any children under 18?" screen
   - Description: Young child reading a book, close-up view
   - Recommended size: 720x200px or similar aspect ratio

## Image Format

- Format: JPG or PNG
- Aspect ratio: ~3.6:1 (landscape, wide)
- Recommended dimensions: 720x200px or higher resolution maintaining aspect ratio
- File size: Optimize for mobile (under 200KB each recommended)

## Fallback

If images are not found, the app will automatically use placeholder gradients with icons.
