# Onboarding Images

Place the following images in this directory for Step 1 (Onboarding) flow:

- `calendar.jpg` - Image for date of birth screen (calendar/date picker)
- `couple.jpg` - Image for spouse screen (couple/marriage)
- `family.jpg` - Image for children screen (family with children)
- `child.jpg` - Image for minors screen (children/guardians)

**Image Requirements:**
- Format: JPG or PNG
- Recommended size: 800x600px or larger
- Aspect ratio: 4:3 or 16:9
- File size: Under 500KB each for optimal performance

The app will automatically use these images when available, and fall back to colored placeholders if they don't exist.
